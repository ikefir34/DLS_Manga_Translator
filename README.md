       За UI в проекте отвечала библиотека flet. После обучения детектора, следующим шагом стало создание 
      первоначальной визуальной оболочки, а именно кнопок загрузки и перегартования страниц, поля для
      отображения картинок. После была импортирована библиотека paddleocr и необходимые библиотеки для 
      API Gemma-3-27B-it. Gemma-3-27B-it была взята с ArliAI по той же причине что и Roboflow, а именно
      недостаток ресурсов вычислительной машины. Для закрашевания текста и вставки переведенного были взяты
      OpenCV и Pil.
       После загрузки изображений в пространство, они сразу сжимаются до размера 1024 на 1024 для обработки 
      YOLO, yolo-разметка как и вся остальная сохраняется в папках для экономии ресурсов. После получения 
      разметки изображения вырезаются и попадают в PaddleOCR. Результаты от PaddleOCR сохраняются в папки 
      по пути storage\paddleocr_annotation\{имя изображения}\{индекс yolo-разметки}. Туда же сохраняется 
      переведенный текст с помощью LLM (Этот момент является краеугольным, так как обработка длится очень долго).

       Как только первоначальная обработка заканчивается дается возможность управления UI. Для UI Были 
      реализованы следующие функции:
        1. Загрузка множественная и одиночная
        2. Сохранение
        3. Перевод с языков, читающихся справа на лево и наоборот
        4. Переключение отображения разметки
        5. Изменение и удаление текста 
        6. Редактирование оформления
      Офомление применяется только на этапе сохранения изображения.

       Инструкция по запуску и упралению:
       1. Запустить файл main.py
       2. Выбрать режим перевода справа налево или слева на право (важно сделать это до начала загрузки,
      ибо перевод работает на режиме который был подключен на этапе загрузки)
       3. Выбрать вариант загрузки
       4. Ждать, долго (если бы ресурсы позволяли, я бы встроил модель в программу)
       5. Можно отредактировать текст в разметке или ее удалить (лучше иметь широкий монитор, так как 
       разметка создается по исходному изображению, а Flet имеет черту сжимать вставленное изображение.
       Потому разметка может слетать)
       6. При изменении текста, не забыть нажать подтвердить
       7. Выбрать стили оформления(цвет поддерживается только HEX)
       8. Нажать на сохранить и выбрать папку для сохранения
